# Arpspoofer

**Requirements:**

* **Root Privileges:** Necessary to send raw network packets.
* **scapy:** Install with `pip install scapy`.

**Usage:**

    ```bash
    chmod +x arp_spoofer.py
    ```

    ```bash
    sudo ./arp_spoofer.py
    ```

4.  **Stop:** Press `Ctrl + C`.

**Important:** add ip address in the target_ip and gateway_ip variables.
